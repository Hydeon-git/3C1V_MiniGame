# 3C1V_MiniGame

Github: https://github.com/Hydeon-git/3C1V_MiniGame

DUCKATTACK is the first game we have created! 
Kill all the red ducks you can by shooting projectiles!

CREATED BY:
David Lira, Adrián Aroca, Albert Pou, Unai Díaz de Garayo.

CONTROLS:
Move your duck by pressing WASD or ⮃⮀ and shoot with the SPACEBAR.
